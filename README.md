# Smart-Expense-Manager-System
A personal finance application to manage expenses and monitor spending.

A simple and efficient web-based Expense Manager application built using Node.js, Express, SQLite, HTML, CSS, and JavaScript. This application helps users track their income and expenses, maintain records, and monitor their financial activities through an easy-to-use interface.

## Features

* Add new income and expense transactions
* View transaction history
* Store data using SQLite database
* Responsive and user-friendly interface
* Real-time expense tracking
* Data persistence using SQLite
* Simple dashboard for managing finances

## Technologies Used

### Frontend

* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* SQLite

## Installation

### Clone the Repository

```bash
git clone https://github.com/ShravaniKale18/Smart-Expense-Manager-System.git
```

### Navigate to Project Directory

```bash
cd Smart-Expense-Manager-System
```

### Install Dependencies

```bash
npm install
```

## Run the Application

Start the server:

```bash
npm start
```

or

```bash
node server.js
```

The application will run locally at:

```text
http://localhost:3000
```

## Usage

1. Open the application in your browser.

3. View and manage transaction records.
